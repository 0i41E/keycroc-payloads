## Description
Play tic-tac-toe / noughts and crosses with the croc. 

Open a text editor, start the game and enjoy.

## Config
Set the language file path on line 15'ish

`lfile=/usr/local/croc/lib/languages/gb.json`

## Match
`(?i)tic[- ]tac[- ]toe`

or

`(?i)noughts (and|\&) crosses`
